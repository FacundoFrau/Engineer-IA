# Agent Review Workflow

## Steps
1. Verificare conformità a `AGENTS.md`.
2. Verificare aderenza a `QUALITY_BAR.md`.
3. Eseguire review su scope/non-scope, I/O, failure modes, validation.
4. Classificare findings (bloccanti/non bloccanti).
5. Registrare esito e remediation.

## Acceptance Criteria
- Nessun finding bloccante aperto.
- Evidenze di verifica disponibili.
- Esito review tracciato.
