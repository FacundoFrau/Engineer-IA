# Agent Creation Workflow

## Steps
1. Definire missione e outcome operativo.
2. Compilare `01_AGENT_DESIGN/AGENT_SPEC_TEMPLATE.md` in modo completo.
3. Modellare task con `TASK_MODEL_TEMPLATE.md`.
4. Definire validazione con `EVALUATION_TEMPLATE.md`.
5. Preparare handoff con `HANDOFF_TEMPLATE.md`.
6. Tracciare decisioni non banali in `00_GOVERNANCE/DECISION_LOG.md`.

## Acceptance Criteria
- Spec completa (nessuna sezione critica vuota).
- Failure modes e validation protocol presenti.
- Acceptance criteria misurabili.
- Decisioni non banali tracciate.
